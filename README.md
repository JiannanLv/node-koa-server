# 基于Node的koa模块服务端

## 安装mysql
```
  1、Win10下的mysql安装文档：https://blog.csdn.net/lvkelly/article/details/102901489
  2、装好mysql以后,创建数据库,代码里的数据库名称：sequelize_study
```

## 安装依赖
```
    npm install 
```

## 修改配置: config
```
    const development = {
        host: 'localhost',
        port: 2019
    };
```

## 启动命令

```
    npm start
```
